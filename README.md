Olá e seja bem vindx ao app que fiz para o processo seletivo da Trinks!

Esse é um aplicativo simples em cli que pede duas coordenadas em um plano cartesiano X e Y, e a partir dele uma direção cardeal e intensidade. O app move a posição atual em direção à instrução cardeal dada, no número de unidades dada pela intensidade.

instruções para rodar:

```bash

#Certifique que tem o node na versão mínima 16 na sua máquina. Em um ambiente com terminal bash, você pode verificar com

node -v

#Copie esse repositório em uma pasta qualquer abrindo o terminal bash nessa e rodando



#instale as dependências com

npm i

#Abra o arquivo desafio.js com node

node desafio.js

#Siga as instruções na tela

```
